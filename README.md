# Magic Apple - NeoForge Mod (Minecraft 1.20.1 / NeoForge 26.1.2)

Magic Apple is a Minecraft mod built using NeoForge.  
This repository contains a clean, minimal project setup with no example items, blocks, configs or template code.  
The goal is to provide a solid foundation for developing custom features such as magical apples, effects and future gameplay mechanics.

---

## Features (current)
- Clean minimal NeoForge mod setup
- Ready for custom items, blocks, effects and systems

---

## Development Setup

### Requirements
- Java 17+
- IntelliJ IDEA (recommended)
- NeoForge MDK (26.1.2)
- Gradle (wrapper included)

### Getting Started
1. Clone this repository:
- https://github.com/Draitz/MagicApple-NeoForge.git
2. Open the project in IntelliJ IDEA.
3. Let Gradle finish syncing.
4. Run the mod using:
- `runClient`
- `runServer` (optional)

---

## Building the Mod
To build a distributable JAR:
./gradlew build

The output will be located in:
build/libs/
